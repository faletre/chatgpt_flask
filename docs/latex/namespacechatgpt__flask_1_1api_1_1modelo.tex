{\footnotesize\ttfamily def actualizar\+\_\+modelo (\begin{DoxyParamCaption}\item[{}]{id }\end{DoxyParamCaption})}



Actualiza el modelo asociado a una conversación específica. 

Este endpoint permite actualizar el modelo de una conversación almacenada en la base de datos. Se valida que el nuevo modelo sea uno de los permitidos ({\ttfamily gpt-\/3.\+5-\/turbo} o {\ttfamily gpt-\/4}).


\begin{DoxyParams}{Parameters}
{\em id} & Identificador único de la conversación. \\
\hline
{\em modelo} & El modelo a actualizar, que debe ser uno de los modelos permitidos.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}

\begin{DoxyItemize}
\item 200 OK\+: Si el modelo se actualiza correctamente, devuelve un mensaje de éxito y el nuevo modelo.
\item 400 Bad Request\+: Si el modelo proporcionado no es válido.
\end{DoxyItemize}
\end{DoxyReturn}
\begin{DoxyNote}{Note}
Los modelos permitidos son\+: {\ttfamily gpt-\/3.\+5-\/turbo} y {\ttfamily gpt-\/4}. Si se intenta actualizar con un modelo diferente, se devuelve un error 400.
\end{DoxyNote}

\begin{DoxyExceptions}{Exceptions}
{\em Exception} & Lanza una excepción si ocurre un error al actualizar la base de datos.\\
\hline
\end{DoxyExceptions}
\begin{DoxyPrecond}{Precondition}
La conversación debe existir en la base de datos con el ID proporcionado. 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
El modelo de la conversación será actualizado con el nuevo modelo.
\end{DoxyPostcond}

{\footnotesize\ttfamily def obtener\+\_\+modelo (\begin{DoxyParamCaption}\item[{}]{id }\end{DoxyParamCaption})}



Recupera el modelo asociado a una conversación específica. 

Este endpoint permite obtener el modelo que está asociado a una conversación en la base de datos. La búsqueda se realiza mediante el ID de la conversación.


\begin{DoxyParams}{Parameters}
{\em id} & Identificador único de la conversación.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}

\begin{DoxyItemize}
\item 200 OK\+: Si la conversación se encuentra, devuelve el modelo asociado.
\item 404 Not Found\+: Si no se encuentra una conversación con el ID proporcionado.
\end{DoxyItemize}
\end{DoxyReturn}
\begin{DoxyNote}{Note}
Si el ID de la conversación no existe en la base de datos, se retornará un error 404.
\end{DoxyNote}
